🚀 النشر في 5 خطوات:

1️⃣  الـ GitHub

   cd c:\Users\عبدالهادي\Desktop\apas-vercel
   
   git init
   git add .
   git commit -m "APAS ready for Vercel"
   
   ثم أنشئ repository على GitHub وألصق الأوامر

2️⃣  Vercel

   اذهب إلى https://vercel.com
   New Project → Import GitHub Repo
   اختر الـ repo الذي ما أنشأته

3️⃣  API Key الجديد

   ⚠️  API Key القديم معروض - يجب تغييره!
   
   اذهب: https://console.anthropic.com
   احذف API Key القديم
   أنشئ API Key جديد
   انسخه

4️⃣  Environment Variables

   في لوحة Vercel:
   Settings → Environment Variables
   
   أضف:
   Name: ANTHROPIC_API_KEY
   Value: (الـ API Key الجديد)

5️⃣  Deploy!

   اضغط Deploy في Vercel
   أو انتظر auto-deploy عند git push
